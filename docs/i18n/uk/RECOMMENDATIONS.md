### Розумні звʼязки

#245 і #551 змістовно повʼязані (Codex-скан): #245 — це чиста дія на рівні
акаунту, тоді як #551 потребує власного стратегічного рішення
(реактивувати/згорнути/замінити).

Оцінка: **Релевантність** = важливість для дорожньої карти/користувачів,
**Складність** = оцінені зусилля на реалізацію, **Модель/Зусилля** =
рекомендована модель Claude і рівень reasoning-effort для реалізації
Claude Code.
