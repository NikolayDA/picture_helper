### Рекомендовано далі (порядок PR)

1. **#564+#568** — базова логіка без Qt спершу й паралельно (незалежні,
   готові до PR).
2. **#565+#569** — інтеграція меню/діалогу по групі після мерджу базової
   логіки (замоканого шляху достатньо).
3. **#566+#570** — перевірка при старті / з'єднання з warmup; #570 додатково
   потребує новий хук `cancel_warmup()` (коментар власника 2026-07-13).
4. **#567+#571** — завершення документації по групі (XS, тривіально).
5. **#551** — отримати рішення щодо стратегії скану (пов'язано з #245),
   потім скоригувати workflow.
6. **#245** — лишається заблокованим зовнішньо; перевірити вручну лише
   після відновлення квоти OpenAI.

відкритих задач, а не переносити його (#542 → #549 мали той самий off-by-one).
