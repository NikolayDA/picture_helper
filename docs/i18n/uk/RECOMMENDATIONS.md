## Поточний стан (2026-07-09)

Активний список аналізу коду порожній. Ruff, mypy і локальний набір тестів
залишаються базою перед новими PR. Від знімка 2026-07-06 закрито всі три
follow-up редизайну **#499/#500/#501** (PR #504) та знахідку мертвого коду
**#503** (PR #506; #505 був випадковим злиттям з порожнім diff, вміст
приїхав через #506); додатково — поліровка іконок/рядка стану
**PR #507/#508**. Нові: баг UI **#509** (курсор інструмента ігнорує zoom
полотна) та **#510** (це оновлення знімка). GitHub наразі показує **13**
відкритих завдань.

- **Ядро редизайну, rail/zoom, картковий інспектор, Dark Mode:**
  **#413/#414/#455–#464/#474–#489** прийшли через PR
  #412/#423/#466/#467/#473/#482/#489; **#490** і **#433/#434** так само
  (епік **#426** тепер залежить лише від **#435**).
- **Закрито з 2026-07-06:** **#499/#500/#501** (PR #504 — світлу схему

### Нове з часу останнього огляду

- **#509 🟠:** курсор пензля/гумки не масштабується зі zoom полотна —
  показаний розмір ≠ фактична площа дії (стосується й пензлів висоти;
  причину точно локалізовано в `set_tool`/`set_brush_size`).
- **#510 🟢:** знімок тріажу випередив PR #504, злитий через 30 хвилин —
  вирішено цим оновленням.

### Ще відкрито

- **O1 🟠 — Додаткові мови середовища виконання.** DE/EN перемикаються;
  es/fr/uk/zh ще бракує як локалей виконання (збігається із **#430**).
- **O8 🟢 — Неточність прототипу:** інструменти висоти лишаються
