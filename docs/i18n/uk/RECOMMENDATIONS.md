1. **#200 виконано (PR #209)** — `setuptools>=78.1.1` запінено у `pyproject.toml` (`[build-system]`) **та** `constraints.txt`; CRITICAL RCE закрито.
2. **#201 виконано (PR #209)** — `wheel==0.46.2` запінено у `constraints.txt`; об'єднано з #200 в один PR пінінгу ланцюга постачання.
3. **#202 виконано (PR #211)** — `pip>=26.1.2` примусово застосовано в кроках setup CI, у хуку SessionStart + dev-доці встановлення; батч CVE (path traversal/symlink/захоплення модулів) закрито.

6. **#166 виконано (PR #219)** — англійські docstrings/коментарі перекладено німецькою по всьому пакету; коментар «без власної копії» уточнено.

9. **#205/#206 виконано (PR #222)** — чисті піни заморожено CVE-регресійними тестами, хук SessionStart встановлює з constraints; issues закрито.
