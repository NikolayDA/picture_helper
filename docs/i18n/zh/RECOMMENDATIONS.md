- **O8 🟢 — 原型不准确：** 高度工具在生成后仍锁定在模拟原型中；仅影响模拟，不影响真实应用（#347）。

4. **#597 + #598** —— 本轮最快的覆盖率收益，两个测试草案都已写在议题中；可合并为一个 PR 完成。
