### 接下来推荐

1. 首先解决 **#656**（设置仓库密钥 `ANTHROPIC_API_KEY`）—— 这是解锁 #646 → #639 → #595 → #582 整条链路仅剩的一个杠杆点。

3. 视觉一行转绿后关闭 **#646**；这会级联关闭 **#639**、**#595** 与 **#582**（手动关闭前逐一简单复核）。
4. **#245** 继续作为纯外部账单/配额跟踪项保留；代码库中无需也无法采取任何行动。
5. 目前**没有**任何未结议题足以支撑一次新的代码 PR —— 下一个合理的 Agent 任务是 #656 完成后的核实工作，而非新的实现。
